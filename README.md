Express start project with Webpack bundler
==========================================

Сайт рок-группы с радио.


Deploy
------

Установка зависимостей npm packages

    $ npm install

Запуск сервера
--------------

    // navigate to localhost:8080 for local dev
    $ npm start

Cборка
------

Сборка статики для разработки

    $ npm run buildDev

 Сборка статики в продакшен

    $ npm run buildProd
